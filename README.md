# 🏥 Hospitalito — Sistema de Gestión Hospitalaria

Sistema de gestión hospitalaria desarrollado en Java aplicando los principios de la **Programación Orientada a Objetos (POO)**.

![HTML](https://img.shields.io/badge/HTML-12.3%25-E34F26?style=flat&logo=html5&logoColor=white)
![POO](https://img.shields.io/badge/Paradigma-POO-6A5ACD?style=flat)

---

## 📌 Descripción

Aplicación que simula el funcionamiento básico de un hospital, permitiendo gestionar pacientes, médicos y citas. Desarrollado aplicando conceptos de POO como herencia, encapsulamiento, polimorfismo y abstracción.

---

## 🧠 Conceptos de POO aplicados

| Concepto | Aplicación en el proyecto |
|---|---|
| Encapsulamiento | Atributos privados con getters y setters |
| Herencia | Clases base como `Persona` extendida por `Paciente` y `Médico` |
| Polimorfismo | Métodos sobreescritos según el tipo de usuario |
| Abstracción | Clases e interfaces que modelan entidades del hospital |

---

## ✨ Funcionalidades

- 🧑‍⚕️ Registro y gestión de pacientes
- 👨‍⚕️ Gestión de médicos y especialidades
- 📅 Programación de citas
- 🗂️ Organización por áreas del hospital

---

## 🛠️ Tecnologías

- **Java** — Lógica principal y modelo de objetos
- **HTML** — Interfaz de visualización
